# Учебные эксперименты на C (CS50)

Этот репозиторий — моя рабочая тетрадь во время прохождения курса CS50.  
Здесь я храню код, пытаясь разобраться в основах алгоритмов, памяти и структур данных.

**Что здесь можно найти:**
- Решения задач, которые я придумываю по мотивам лекций.
- Эксперименты с рекурсией, итерациями, указателями.
- Вспомогательные библиотеки, которые я собираю по ходу обучения.

**Важно:** этот код написан **для себя**. Он не претендует на идеальность или готовность к использованию. Это черновики, которые показывают, как я думаю и как учусь.

## Моя цель (уже невыполняется долгое время)

- Каждый день писать хотя бы несколько строк кода.
- Пробовать разные подходы, ошибаться, исправлять.
- Со временем возвращаться к старым файлам и улучшать их, видя свой прогресс.

## Если вы сюда заглянули
